# Vibrant Tara Bala

A powerful Vedic astrology web application for exploring:

- Tara Bala cycles
- Live Moon transit tracking
- Nakshatra-based analysis
- Moon Pada transitions
- Birth chart cloud fetching
- Custom Tara finder
- 30-day Moon transit explorer
- Progressive Moon syncing
- Live planetary transit calculations

---

## Features

### 🌙 Live Moon Transit Engine
- Real-time Moon Nakshatra tracking
- Exact Pada transition timings
- Tara Bala calculation against Janma Nakshatra
- Current transit highlighting

### 🔮 Tara Bala Matrix
- Full 3-cycle Tara chart
- Janma to Param-Mitra classification
- Planetary lord mapping
- Dynamic Tara categorization

### 📡 Cloud Birth Fetcher
- Auto-fetch birth chart data
- Profile save/load support
- GPS-based location support
- City search integration

### 🗺️ Interactive Location System
- OpenStreetMap search
- Preset locations
- Manual GPS input
- Live coordinate syncing

### 📅 Moon Transit Explorer
- 30-day transit scanning
- Entry & exit timings
- Rasi transitions
- Nakshatra & Pada tracking

### ⚡ Advanced UI
- Neon vibrant astrology theme
- Fully responsive layout
- Animated sync indicators
- Live status banners
